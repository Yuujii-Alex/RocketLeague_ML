# Shen RLBot

This is a project born out of interest for **Machine Learning** and **Rocket League**.

The goal is to train a competitive Reinforcement Learning agent using state-of-the-art techniques and observing how it evolves from basic movement to high-level strategic play.

## Technology Stack

- **Engine**: [RocketSim](https://github.com/ZealanL/RocketSim) (via RLGym 2.0)
- **Framework**: [RLGym-PPO](https://github.com/AechPro/rlgym-ppo)
- **Algorithm**: Proximal Policy Optimization (PPO)
- **Environment**: RLGym 2.0
